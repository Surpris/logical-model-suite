# Suite of Logical Models

This repository contains a suite of logical models for various purposes.

## Purpose

- To provide a schema definition of logical data models.
  - The schema is defined in JSON Schema.
  - The logical data models are defined in YAML.
- To provide a reference implementation of logical data models.
  - The reference implementation is defined in TypeScript.
- To provide tools:
  - to validate logical data models.
  - to generate typescript interfaces from logical data models.
  - to generate Prisma schema from logical data models.
  - to generate GraphQL schema from logical data models.
